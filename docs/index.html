Finder_bot est un assistant intelligent qui utilise le traitement du langage naturel et les données géospatiales pour vous localiser avec précision à partir de simples descriptions.

Explorer le code En savoir plus

Pourquoi Finder_bot ?

Intelligence Agentique

Propulsé par un LLM (Ollama), le bot comprend vos intentions et décide de la meilleure stratégie de localisation.

Recalage par Repères

Dites ce que vous voyez (monuments, commerces, rues) et le bot ajuste instantanément votre position sur la carte.

Estimation de Trajet

À partir de votre point de départ, destination et durée de trajet, Finder_bot calcule mathématiquement votre position actuelle.

Données OpenStreetMap

Exploitation massive de données réelles via Nominatim, OSRM, Overpass et OSMnx pour une précision maximale.

Comment ça marche ?

1

Dialogue Naturel

L'utilisateur décrit sa situation : "Je suis parti de la Tour Eiffel vers le Louvre il y a 10 minutes à pied."

2

Extraction d'Entités

Le LLM extrait les points clés : départ, arrivée, mode de transport, durée écoulée.

3

Calcul & Raffinement

Le système calcule une position probabiliste et propose des points d'intérêt (POI) proches pour confirmation.

4

Visualisation

La position est projetée sur une carte interactive Leaflet avec un tracé d'itinéraire dynamique.
